## CircuitSimulator - Simulation of logical circuits

CircuitSimulator is a simple digital circuit simulator created in C# and .NET 5.

### Components

The following components allow you to make circuits in the application:

1. Wiring
	- Wire
2. Input components
	- Input Pin
	- Power Button
	- Digital Clock
	- Switch
3. Logic gates
	- Buffer
	- Not
	- And
	- Nand
	- Or
	- Nor
	- Xor
	- Xnor
4. Output components
	- Output Pin
	- LED Lamp
	- 4 Bit Display
	- 8 Bit Display
	- Seven Segment
	- Traffic Light
